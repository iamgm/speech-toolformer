# Ты — Senior Data Engineer. Твоя задача — сгенерировать синтетический датасет для обучения AI-ассистента "Magic Paste"

Этот ассистент помогает пользователю редактировать текст или писать новый прямо в активном окне приложения.

СТРУКТУРА ИНСТРУМЕНТА:
Ассистент имеет доступ к одному инструменту:
Name: magic_paste
Arguments: text (строка с финальным готовым текстом)

ФОРМАТ ОТВЕТА МОДЕЛИ:
Если пользователь просит написать, переписать, исправить или дополнить текст — ассистент ДОЛЖЕН вернуть XML блок:

```xml
<tool_call>
<name>magic_paste</name>
<arguments>
<text>Здесь находится готовый текст</text>
</arguments>
</tool_call>
```

Если пользователь просто здоровается, задает отвлеченный вопрос (не про генерацию текста) — ассистент отвечает обычным текстом БЕЗ вызова инструмента.

ЗАДАНИЕ:
Сгенерируй 60 обучающих примеров на РУССКОМ языке в формате JSONL.
Распределение:

1. 20 примеров: Рерайтинг (Пользователь дает грязный текст и просит сделать формально/вежливо/кратко).
2. 20 примеров: Генерация с нуля (Напиши письмо, создай функцию на Python, придумай поздравление).
3. 20 примеров: Chit-chat / Отказ (Привет, как дела? Какая погода? — здесь инструмент вызывать НЕЛЬЗЯ).

ФОРМАТ ВЫВОДА (JSONL, каждая строка — валидный JSON object):
{"messages": [{"role": "user", "content": "..."}, {"role": "assistant", "content": "..."}]}

ВАЖНО:

- Тексты внутри `<text>...</text>` должны быть качественными и готовыми к вставке.
- Не добавляй никаких пояснений перед JSONL. Только сырые данные.
